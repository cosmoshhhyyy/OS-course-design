    使用说明

    • mkdir 新建目录
      •     mkdir rot    mkdir rot/aa    mkdir rot/bb    mkdir rot/cc
    • touch 新建文件
      •     touch rot/c.ex ( start;x=1;x++;x--;?a9;end; )  touch rot/aa/d.ex ( start;x=3;x++;x--;?a5;end; )  touch rot/bb/f.ex ( start;x=3;x++;x--;?c5;x--;x++;end; )  touch rot/g.ex ( start;x=8;x++?b6;x--;x++;end; )
          touch rot/cc/r.tx ( abcdefghigklmn )   touch rot/bb/t.tx ( 123456789 )
    • cat vi 查看文件(也可修改文件)
      •     cat rot/c.ex    vi rot/cc/r.tx
    • cp 拷贝文件
      •     cp rot/bb/t.tx    rot/cc/t.tx
    • mv 移动文件
      •     mv rot/bb/t.tx    rot/aa/t.tx
    • rm 可以删除文件或目录
      •     rm rot/cc
    • .ex文件可加入就绪队列，启动cpu可开始运行
      •     rot/c.ex    rot/aa/d.ex    rot/bb/f.ex
    • 注意事项(用前必读)
      • 文件名与目录名最长3个字母，拓展名最长2字母
      目录和文件要一层一层新建，不能递归创建
      在输入文件名时要带上拓展名
      一个目录最多有8个子目录或子文件
      拷贝或移动文件时，写上预期文件名(改名,可以选择不改，加上原有名即可)
      只有包含拓展名的才是文件，图标只表示目录中有无子文件
